## int 和 integer 相同吗？

int 是基本数据类型，int 变量存储的是数值。Integer 是引用类型，实际是一个对象，Integer 存储的是引用对象的地址。[Java 中 int 和 Integer 的区别](https://www.runoob.com/note/37965)
